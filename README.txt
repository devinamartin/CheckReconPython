To use have 2 files named:
ManualDeposit.xlsx
SFDeposit.xls

These imported files mimic generated reports that I would use at work and review manually, this script
was designed to reduce research time and narrow down inconsistencies with multiple data entry users.

The main function is to compare check sums to either one-to-one matches or a set of split items (A check deposit split between multiple accounts)
